# 808 Sample Pack

Roland TR-808 drum machine one-shots.

These WAV files are intended for the `load wav` sampler command.

Example:

```text
vcpi> load wav 2 808 bassdrum
```

This resolves to `samples/808/bassdrum.wav`.

Included one-shots:

- `bassdrum.wav`
- `snare.wav`
- `hihat-closed.wav`
- `hihat-open.wav`
- `clap.wav`
- `cowbell.wav`
- `cymbal.wav`
- `claves.wav`
- `rimshot.wav`
- `maracas.wav`
- `conga-high.wav`
- `conga-mid.wav`
- `conga-low.wav`
- `tom-high.wav`
- `tom-mid.wav`
- `tom-low.wav`

See `samples/808/ATTRIBUTION.txt` for source details.
