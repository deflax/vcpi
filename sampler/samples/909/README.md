# 909 Sample Pack

These WAV files are intended for the `slot <n> wav` sampler command.

Example:

```text
vcpi> slot 2 wav 909 bassdrum
```

This resolves to `samples/909/bassdrum.wav`.

Included one-shots:

- `bassdrum.wav`
- `kick.wav`
- `kick-short.wav`
- `snare.wav`
- `hihat.wav`
- `hihat-closed.wav`
- `hihat-open.wav`
- `clap.wav`
- `rimshot.wav`
- `ride.wav`
- `cymbal.wav`
- `tom-high.wav`
- `tom-low.wav`

See `samples/909/ATTRIBUTION.txt` for source details.
